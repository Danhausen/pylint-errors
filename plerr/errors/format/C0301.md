## C0301 (line-too-long)

### :x: Problematic code:

```python
# to be addded
```

### :heavy_check_mark: Correct code:

```python
# to be added
```

### Rationale:

 *Line too long (%s/%s)*
  Used when a line is longer than a given number of characters. This message
  belongs to the format checker.
