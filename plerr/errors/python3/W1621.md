## W1621 (dict-view-method)

### :x: Problematic code:

```python
data = {'x': 1}
print(data.viewitems())
```

### :heavy_check_mark: Correct code:

```python
data = {'x': 1}
print(data.items())
```

### Rationale:

Used for calls to `dict.viewkeys()`, `viewvalues()` or `viewitems()` (Python 3
lacks these methods).
