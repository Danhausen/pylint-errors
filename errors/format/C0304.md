## C0304 (missing-final-newline)

### :x: Problematic code:

```python
# to be addded
```

### :heavy_check_mark: Correct code:

```python
# to be added
```

### Rationale:

 *Final newline missing*
  Used when the last line in a file is missing a newline. This message belongs
  to the format checker.
