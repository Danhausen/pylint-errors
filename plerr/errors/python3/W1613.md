## W1613 (xrange-builtin)

### :x: Problematic code:

```python
for i in xrange(100):
    print(i)
```

### :heavy_check_mark: Correct code:

```python
for i in range(100):
    pass
```

### Rationale:

Used when the xrange built-in function is referenced (missing from Python 3).
