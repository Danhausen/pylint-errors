## C0198 (bad-docstring-quotes)

### :x: Problematic code:

```python
# to be addded
```

### :heavy_check_mark: Correct code:

```python
# to be added
```

### Rationale:

 *Bad docstring quotes in %s, expected """, given %s*
  Used when a docstring does not have triple double quotes. This message
  belongs to the docstyle checker.
