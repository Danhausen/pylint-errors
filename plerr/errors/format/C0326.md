## C0326 (bad-whitespace)

### :x: Problematic code:

```python
# to be addded
```

### :heavy_check_mark: Correct code:

```python
# to be added
```

### Rationale:

 *%s space %s %s %s*
  Used when a wrong number of spaces is used around an operator, bracket or
  block opener. This message belongs to the format checker.
