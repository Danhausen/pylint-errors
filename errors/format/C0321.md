## C0321 (multiple-statements)

### :x: Problematic code:

```python
# to be addded
```

### :heavy_check_mark: Correct code:

```python
# to be added
```

### Rationale:

 *More than one statement on a single line*
  Used when more than on statement are found on the same line. This message
  belongs to the format checker.
