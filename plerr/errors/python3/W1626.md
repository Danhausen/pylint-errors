## W1626 (reload-builtin)

### :x: Problematic code:

```python
import third_party

reload()
```

### :heavy_check_mark: Correct code:

```python
import importlib


import third_party

importlib.reload()
```

### Rationale:

Used when the `reload` built-in function is referenced (missing from Python 3).
You can use instead `imp.reload` or `importlib.reload`.
